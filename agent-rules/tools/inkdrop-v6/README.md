# Inkdrop v6 Agent Rules

Use this rule pack when creating, updating, organizing, or auditing Inkdrop notes.

## Routing

- Mermaid diagrams: read `mermaid.md`.
- Internal links, backlinks, indexes, or note references: read `note-links.md`.
- Notebook, tag, status, and note-type organization: read `organization.md`.
- MCP search/read/update token efficiency: read `mcp-token-workflow.md`.
- General Markdown formatting: read `../../shared/markdown.md`.
- Researched claims or version-sensitive docs: read `../../shared/research-citation.md`.

## Default Workflow

1. Search/list first; read full notes only when needed.
2. Pick the durable notebook, note type, status, and tags before writing.
3. Draft concise body content without duplicating native metadata.
4. Validate Mermaid and internal links before create/update.
5. Report changed note titles, note IDs, notebook, status, tags, and unresolved manual actions.
